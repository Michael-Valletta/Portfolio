Dreadonic (In Development)
---------------------------

Dreadonic is a 3D Survival/Horror game where players take on the role of a Vampire Knight. The core gameplay loop focuses on realistic combat, RPG-style character progression (leveling/gearing), and high-stakes Boss encounters.

Current Technical Focus:
--------------------------

Engine: Unreal Engine 5.

Status: Early Alpha / Prototyping.

Implemented Features: Player Movements, Main Menu UI , Mini Map , Map Lighting, Background Music, First Boss Fight Music

Note: As the project is in active development, the source code is currently private. In this File are screenshots showcasing our progress so far in game. Some items are expected to change overtime.
