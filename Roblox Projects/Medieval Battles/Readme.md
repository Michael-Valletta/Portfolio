**Medieval Battles**



-A round-based combat simulator focused on medieval warfare and strategic map selection.



**Key Features \& Systems:**



-Dynamic Map Option/Selection: Scripted a GUI-driven system that allows players to choose the map or opt for a Randomized Selection logic.



-Combat Analytics: Implemented a system to track "Kill/Death" ratios in real-time to encourage competitive play.



-Environment Design: Used distinct medieval-themed maps optimized for performance and player flow.
