**Deadly Run:**



-An endless runner inspired by Temple Run, focused on high-speed reaction and score optimization.



**Key Features:**



-Dynamic Obstacle Spawning: Scripted a system to randomly place hazards to keep gameplay unpredictable.



-UI/UX Design: Built a custom HUD to display real-time score tracking and "Game Over" statistics.



-Character Physics: Adjusted Luau walk-speed variables and jump-power to create a responsive, arcade-like feel.
