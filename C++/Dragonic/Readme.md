#### **Dragonic**



Dragonic is a survival-style action game originally developed as a C++ home assignment, however I continued to polish this project as I liked where it was heading. The project focuses on core game loop architecture, basic AI behaviors, and 2D rendering using the Raylib library.





**Gameplay:**



Survive: Hold out for 10 minutes against waves of enemies.



Level Up: Collect XP drops to choose between Movement Speed or Attack Speed upgrades.



Auto-Battle: The player automatically targets the nearest threat with fireballs.





**Key Technical Features:**



Dynamic Scaling: Enemy spawn rates increase as the timer goes down.



Asset Integration: Managed background music, and sound effects with proper memory de-allocation.



Nearest-Neighbor Logic: A targeting system that iterates through active entities to find and track the optimal target.
