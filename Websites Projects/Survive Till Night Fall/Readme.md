**Survive Till NightFall Website**



-A high-atmosphere landing page developed for a fictional survival-horror title, focusing on immersive UI and interactive elements.



**Tech Stack:**

HTML, CSS, JavaScript



**Key Features \& Technical Implementation:**



Interactive Animations: Leveraged CSS Keyframes and JavaScript event listeners to create a dynamic, "living" interface that reflects the game's dark atmosphere.



Thematic UI/UX: Designed the layout with a "Survival-Horror" aesthetic, ensuring the typography and color palette align with the game’s core pillars.
