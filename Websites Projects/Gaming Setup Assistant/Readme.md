**Gaming Setup Assistant Website**



-A responsive web application designed to help users calculate and optimize the cost of their custom gaming environments.



**Tech Stack:** 

HTML, CSS, JavaScript



**Key Features \& Technical Implementation:**



-Dynamic Quote Engine: Developed a JavaScript-based calculator that processes multiple user inputs (setup tier, monitor count, and region) to generate real-time cost estimates.



-Conditional Logic: Implemented a Setup Tier system where the pricing scales based on specific user requirements.



-Responsive UI: Designed a clean, user-friendly interface using CSS to ensure the assistant works perfectly on desktop.



-Vendor Integration Logic: Integrated logic to adjust pricing based on selected retail regions or currency variables.
